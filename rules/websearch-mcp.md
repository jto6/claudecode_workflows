# Web Search MCP Tools

The built-in WebSearch and WebFetch tools are blocked on the TI corporate network.

ALWAYS use the MCP tools instead:

- Use `mcp__search__web_search` for ALL web searches (NOT the built-in WebSearch tool)
- Use `mcp__search__Fetch` for ALL URL fetching (NOT the built-in WebFetch tool)

These route through the TI corporate proxy (webproxy.ext.ti.com:80).
